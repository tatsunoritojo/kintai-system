# 勤怠管理システム - GitHub Pagesプロトタイプ

このディレクトリには、クライアント確認用のスタティックHTMLプロトタイプが含まれています。

## 📁 構成

```
docs/
├── index.html              # ダッシュボード
├── login.html              # ログイン画面
├── settings.html           # 設定画面
├── css/
│   └── style.css          # カスタムスタイル
├── js/
│   ├── mock-data.js       # モックデータ
│   └── app.js             # ユーティリティ関数
├── employees/             # 従業員管理
├── work-records/          # 勤務記録
├── payrolls/              # 給与計算
├── hourly-wages/          # 単価マスタ
├── students/              # 生徒マスタ
├── work-types/            # 勤務形態マスタ
└── admin/                 # 管理機能
```

## 🚀 ローカルでの確認方法

### 方法1: ブラウザで直接開く
`docs/index.html` をブラウザで開いてください。

### 方法2: ローカルサーバーを起動
```bash
# Pythonがインストールされている場合
cd docs
python -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```

## 🎨 機能

- **ダッシュボード**: 月次統計、最近の勤務記録、通知
- **従業員管理**: 一覧、詳細、登録・編集
- **勤務記録**: 一覧、登録
- **給与計算**: 一覧、計算実行、詳細
- **マスタ管理**: 単価、生徒、勤務形態
- **管理機能**: バッチ実行、設定

## 🔄 ロール切り替え

ナビゲーションバーの「ADMIN」ボタンをクリックすると、管理者モードと従業員モードを切り替えられます。

## 📝 注意事項

- これはプロトタイプです。実際のデータベース接続やAPI通信は行いません。
- すべてのデータは `js/mock-data.js` に定義されたモックデータです。
- フォーム送信やボタンクリックは、アラート表示のみで実際の処理は行いません。
