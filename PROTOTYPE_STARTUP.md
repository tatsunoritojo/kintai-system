# プロトタイプ起動手順

## 前提条件

- Java 17がインストールされていること
- Mavenがインストールされていること

## 起動方法

### 方法1: Mavenコマンドで起動（推奨）

```bash
# プロジェクトルートディレクトリで実行
mvn spring-boot:run
```

### 方法2: jarファイルをビルドして起動

```bash
# ビルド
mvn clean package -DskipTests

# 起動
java -jar target/attendance-management-system-0.0.1-SNAPSHOT.jar
```

## アクセス方法

起動後、ブラウザで以下にアクセス:

- **ログイン画面**: http://localhost:8080/login
- **ダッシュボード**: http://localhost:8080/dashboard
- **従業員一覧**: http://localhost:8080/employees

## トラブルシューティング

### ポート8080が既に使用されている

`application.yml` で別のポートに変更してください:

```yaml
server:
  port: 8081
```

### ビルドエラーが発生する

依存関係をクリーンインストール:

```bash
mvn clean install
```

### 画面が表示されない

ブラウザのキャッシュをクリアしてください（Ctrl + Shift + R）。
